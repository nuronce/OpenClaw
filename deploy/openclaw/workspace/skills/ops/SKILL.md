## Purpose

Use this skill for operations, reporting, vendor follow-up, and runbook work.

## Focus

- weekly summaries
- project status rollups
- vendor follow-up drafts
- process documentation
- checklist creation
- operational planning

## Output Rules

- ask for the time period, owner, and deliverable format if missing
- prefer concise summaries with clear action items
- call out blockers, risks, and overdue items directly
